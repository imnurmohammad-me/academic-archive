Admin App User Manual

1.Introduction

Restart is a versatile taxi and on-demand ride booking application that not only provides transportation and ride-sharing services but also facilitates parcel delivery and logistics solutions. Restart offers clients all the essential services, including the ability to book various types of vehicles, set booking parameters, and access comprehensive booking details.

Our application supports multiple currencies and time zones, making it suitable for deployment worldwide. Additionally, Restart features multilingual support, allowing users to interact in their preferred languages. You can customize localization content through the translation options available in our admin app, ensuring a tailored experience for users across different regions.

1.Dashboard & Owner Dashboard

The dashboard serves as a comprehensive control center for managing your entire application. It provides both Admin and Owner users with a detailed overview of essential metrics and performance indicators.

Key features include:

  • User and Driver Management: Easily monitor the number of registered users and active drivers within the application. This feature helps in assessing user engagement and driver availability.
  • Trip Overview: Gain insights into current trip statistics, including the number of ongoing rides, completed trips, and canceled rides. This data assists in evaluating the efficiency of operations.
  • Earnings Overview: View financial summaries for the current year, including total earnings, breakdown of revenues from rides and deliveries, and insights into any discounts or promotions applied. This is vital for financial planning and strategy.
  • Real-time Alerts: Receive notifications for critical events such as new user registrations, driver applications, and any issues that may arise in the system. This ensures that you remain proactive in managing your operations.
  • Performance Metrics: Access key performance indicators (KPIs) to assess the overall health of the application, helping you identify areas for improvement and growth opportunities.

Overall, the dashboard empowers both Admins and Owners with the insights needed to effectively manage and optimize their ride-hailing and logistics services.

2.Dispatcher Module

The Dispatcher Module enables admins to manage ride bookings directly through the panel. Dispatchers can select a pickup and drop-off location for the ride and assign a driver manually. This feature allows for more precise control over the ride scheduling and driver assignment, ensuring efficient ride management.

Additionally, dispatchers can schedule rides in advance, enabling flexibility for both drivers and riders. This feature supports various use cases, such as planned rides.

3.Price Management

Price Management is the foundational setup for managing the application, allowing you to configure service locations, zones, vehicle types, goods types, subscriptions, and driver incentives. Each of these options is essential for tailoring your service offerings, managing users and drivers, and setting up pricing models across different regions and service types.

Each option includes detailed sub-menus with visual guides and explanations, as listed below:

  • Service Location: Define and manage service areas where your application will be available. Set unique currencies and time zones for each location.
  • Zone Management: Create zones within service locations to define specific areas for price variations and restrictions. This feature helps segment areas for more customized pricing.
  • Vehicle Type: Define different vehicle types for the service, such as cars, bikes, or trucks, and set base prices for each type.
  • Goods Type: Define and manage different types of goods that can be transported, with the option to set unique pricing for each type.
  • Subscriptions: Offer subscription packages for users or drivers, allowing them to access exclusive benefits or lower rates.
  • Driver Incentives: Set up incentive programs for drivers based on various factors, such as completed rides or user ratings, encouraging high-quality service.

4.Geofencing

The Geofencing module provides two unique views for monitoring driver availability and ride demand across service areas: "God's Eye" and "Heat Map." These views allow dispatchers and admins to gain real-time insights into driver distribution and areas of high demand, enabling informed pricing adjustments and resource allocation.

Geofencing Views

In the Geofencing view, you can choose between the following options:

God's Eye

The "God's Eye" view displays the real-time status of drivers within the service area, showing available and busy drivers. Driver markers update in real-time using Geofire, giving dispatchers a comprehensive overview of driver availability at any moment.

Heat Map

The "Heat Map" view shows areas with high demand for rides, represented by color intensity. This tool helps identify peak areas and times, allowing admins to adjust pricing dynamically based on demand and improve service coverage.


5.Admin, Driver, User & Owner Management

Admin Management

In the Admin Management menu, you can create new admin users and assign them to specific service locations. The super admin has the ability to create new super admin users as well. Each admin user can create additional admin users with varying privileges tailored to their roles within the system.

User Management

In this menu, you can view the list of registered users who are actively using our application. Admins can edit user information and delete user accounts as necessary.

Driver Management

The Driver Management menu displays a comprehensive list of all registered drivers using our application. Admins can edit driver information and delete driver accounts. Additionally, there is a document view option in the drivers list to manage driver documents. Admin users can upload and remove documents for each driver, approve or decline documents, and provide comments during the decline process. These comments will be visible in the mobile approval screen.

Manage Documents

Owner Management

The Owner Management section allows you to oversee fleet operations and manage fleet drivers under each owner. This feature is designed to streamline the administration of vehicle fleets and their assigned drivers, ensuring efficient oversight of resources.

Fleet Management

Under Fleet Management, owners can view and manage their entire fleet of vehicles, including details about each vehicle's status and availability.

Fleet Drivers

In this submenu, owners can view all drivers associated with their fleet. This section provides insights into driver activities, including approved and declined drivers, along with driver statistics to monitor performance effectively.
