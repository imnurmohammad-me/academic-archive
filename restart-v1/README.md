# Restart Project Ecosystem 🚀

This repository contains the complete "Restart" app ecosystem, including the administrative backend and mobile applications for users and drivers.

## 📂 Components

- **[Admin Panel](./Main-File-March-31/Admin-Panel-1.5)**: The central management system built with Laravel.
- **[User App](./Main-File-March-31/user_envato)**: The mobile application for customers (Flutter).
- **[Driver App](./Main-File-March-31/driver_envato)**: The mobile application for service providers/drivers (Flutter).

## 🪜 General Setup

1. **Backend**: Setup the Laravel Admin Panel first (requires PHP, MySQL, and Composer).
2. **Mobile Apps**: Configure the Flutter apps to point to your hosted Admin Panel API.

See the README in each sub-directory for specific instructions.
