## Purpose

This repository is an organized archive of academic work completed during university. It exists to keep the main portfolio focused on production-ready software while preserving the historical record of foundational learning and experimentation.

> All code here is for archival and academic reference only. It is not actively maintained.

---

## Contained Projects

| Project | Description | Stack |
|---|---|---|
| `cvpr_spring_23_24` | Computer Vision and Pattern Recognition coursework from Spring 2023-24. Image processing algorithms and ML model experiments. | Python, OpenCV |
| `lab-1` | Introductory data structures and algorithms laboratory exercises. | Java |
| `login-registration-validation` | A legacy PHP/MySQL authentication prototype demonstrating secure password hashing and SQL injection prevention before modern frameworks. | PHP, MySQL |
| `restart-v1` | An archived early-stage project template used as a starting base. | Mixed |
