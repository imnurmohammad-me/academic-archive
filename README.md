# Academic & Legacy Archive

This monorepo serves as an archive for early academic assignments, lab coursework, and legacy prototype systems. It was established to keep the main digital portfolio clean and focused on production-ready SaaS products.

---

## 🏛 Contained Projects

### 1. `cvpr_spring_23_24`
Computer Vision and Pattern Recognition (CVPR) coursework and laboratory assignments from the Spring 23-24 semester. Includes foundational image processing algorithms and machine learning models.

### 2. `lab-1`
An introductory laboratory repository containing foundational programming exercises and data structure implementations.

### 3. `login-registration-validation`
A legacy PHP/MySQL authentication prototype. It demonstrates custom form validation, SQL injection prevention techniques (prepared statements), and secure password hashing before modern frameworks were adopted.

### 4. `restart-v1`
An archived historical project template or early-stage system prototype.
